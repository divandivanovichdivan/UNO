# Игра UNO для командной строки.

Данный проект может быть использован для игры в UNO для нескольких человек на одном устройстве.

### Как установить

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

### Пример запуска

Предупреждение: Не храните важные данные в командной строке, так как в процессе игры она будет очищена!

```
>>>python main.py

Введите количество игроков: >>>3
Первым ходит 2 игрок.
(Нажмите Enter для начала игры)

...

Победил 1 игрок!
```

Для игры достаточно знать правила игры UNO и выполнять действия, запрашиваемые программой.
